System Overview & 18-Week Roadmap · Generated 2026-06-30

1. Problems Vestra solves

Kenya's real-estate market loses billions every year to fraud, opaque pricing, paper leases that vanish, and chaotic rent collection. Vestra closes those gaps with one unified, trust-graded platform.

Buyers

  • Forged title deeds & ghost sellers
  • No transparent price-per-sqft data
  • Paying cash to unverified parties
  • No escrow culture — wire-and-pray

Sellers

  • Cannot reach verified buyers directly
  • No analytics on listing performance
  • Commission opacity with brokers

Landlords

  • Rent across cash / M-Pesa / bank — chaos
  • Word-doc leases over WhatsApp
  • Maintenance lost in phone calls
  • No tax-ready rent roll

Tenants

  • No proof of payment for visa/credit apps
  • Maintenance complaints ignored
  • Lease disputes — paper-only
  • Heavy upfront move-in costs

Agents

  • Leads on WhatsApp / Excel — no CRM
  • Commission disputes with brokerages
  • EARB license not visible to consumers

Investors (Chama)

  • Group purchases tracked on chalkboards
  • No transparent contribution ledger
  • Decisions on phone calls — no voting

2. Differentiators

Trust layer. AI title-deed verification (OCR + Ministry of Lands cross-check), Trust Score 0–100 per property, EARB-verified Agent badges, multi-party escrow wallet.

Multi-role identity. One Vestra account, activate roles as life evolves: Member → Landlord → Seller → Investor → Tenant. Switchable from topbar / sidebar.

Native M-Pesa rails. STK Push for rent, deposits, escrow funding. Auto receipts (PDF) + tax-ready CSV. Paybill auto-reconciliation.

Chama Cooperative Engine. Shared cap table, M-Pesa-anchored contribution ledger, in-app voting, buy-out calculator.

Lease-as-a-Service. Template marketplace + e-sign + hash-anchored timestamps + dispute timeline.

Maintenance Marketplace. Vetted contractor pool, photo-required updates, SLA timer, micro-escrow.

Investment Intelligence. Suburb price index, rental yield calculator, comparable sales w/ confidence intervals.

6. Mobile Path — Play & Apple Stores

Recommended architecture: Capacitor. Wraps the existing Vite/React PWA into native iOS + Android binaries. One codebase, native APIs available, ~2 weeks engineering for the wrap itself, then 6–8 weeks to submission and approval.

6.1 Setup commands

# 1. Install Capacitor
cd vestra_properties
npm install @capacitor/core @capacitor/cli @capacitor/ios @capacitor/android
npx cap init "Vestra" "com.vestra.app" --web-dir=dist

# 2. Build web bundle
npm run build

# 3. Add native platforms
npx cap add ios
npx cap add android
npx cap sync

# 4. Open in native IDEs
npx cap open ios       # Xcode
npx cap open android   # Android Studio

6.3 Store submission checklist

Apple App Store

  • Apple Developer Account ($99/yr)
  • App Store Connect listing — name, subtitle, keywords, screenshots @ 6.5" / 5.5"
  • Privacy policy URL
  • App Privacy questionnaire (PII, location, financial data)
  • TestFlight beta — 25+ Kenya testers
  • Review: 24–72 hr typical

Google Play

  • Play Console ($25 one-off)
  • Short + full description, feature graphic, 8 screenshots
  • Content rating questionnaire
  • Data safety declaration
  • Closed testing — 20+ testers × 14 days
  • Closed → open → production release

6.4 Compliance landmines

  • Financial services flag — Apple may demand banking license disclosure if M-Pesa rent is in-app. Mitigation: position as "third-party payment integration," surface clear T&Cs.
  • Real-estate brokerage license — visible reference to EARB compliance required.
  • KYC privacy — disclose ID collection in Privacy Policy.

9. KPIs & Risks

KPIs to track

  • Activation — % of signups who activate ≥1 non-buyer role within 7 days
  • Subscription conversion — free → paid by role
  • NRR — keep > 100% via tier upgrades + take rate
  • Transaction GMV — escrow + rent processed / month
  • Trust Score avg — > 85 across active listings
  • Maintenance SLA — % tickets closed within stated window
  • DAU/MAU — > 25%
  • Mobile share — > 70% within 6 months of native launch
